'Spaceship Titanic' Kaggle Playground Series

exploring sklearn classifiers

A classifier model that predicts the outcome of passengers in a ship crash. Based on actual data and passenger manifests, we are able to detect patterns that the model can train on, such that it can predict survivability of the remaining missing passengers, so as to best allocate search-and-rescue resources.

Achieved 0.80576 accuracy score on Kaggle, leaderboard position 256/2350.

Techniques used:
conditional imputation of missing values, KNNImputer, OneHotEncoder, OrdinalEncoder, Regex, RobustScaler, PCA, GridSearchCV, GaussianNB, LogisticRegression, SVC, Adaboost, RandomForest, HistGBM, Pipelines (for final model).

'Manufacturing Control Data' Kaggle Playground Series

exploring clustering algorithms

Given manufacturing control data for which each sample belongs to a cluster, the task is to predict cluster labels.

First attempt with unsupervised methods, and observing how different algorithms work/don't work for this dataset.Here, k-Means is too simplistic (creates spherical clusters), dataset density was too uniform and thus density based methods didn't seem to work well either. Gaussian Mixture Models seemed the most accurate thus far. Also attempted a semi-supervised approach, using 'confident' predictions out of the GMM as labels to train a HistGBM classifier. Doing so managed to eke out a little bit more score-wise. 

PCA reduction wasn't useful either, so only utilised it and t-SNE for visualisation.

Techniques used:
Shapiro-Wilks Test, Chi-squared Test, PowerTransformer, clip outliers, PCA, k-Means, GMM, BGMM, Semi-supervised using HistGBM, t-SNE (visualisation)
